# LRTOP

Това е уеб приложение, разработено като Google Apps Script, предназначено за управление на ревизии и инвентаризация в търговски обекти.

## Основни възможности

- Сканиране на баркодове и автоматично попълване на артикули.
- Въвеждане и приключване на ревизии в Google Sheets.
- Генериране на ППР (приемо-предавателен разпис) с възможност за изпращане по имейл.
- Запазване на данни и файлове в Google Drive.
- Вграден интерфейс за преглед на съществуващи ревизии и качени файлове.

## Структура на проекта

- **Code.js** – основният скрипт с Apps Script функции за работа със Spreadsheet и Drive.
- **index.html** – основният интерфейс за сканиране на баркодове и управление на ревизиите.
- **interface.html** – страница за преглед на съдържание на папка в Google Drive.
- **reference.html** – страница за допълнителни операции с файлове и търсене.
- **appsscript.json** – конфигурационен файл за Apps Script проекта.

## Инсталиране

1. Създайте нов проект в [Google Apps Script](https://script.google.com/).
2. Качете всички файлове от това репо в проекта (File > New > Script file / HTML file).
3. Отворете `appsscript.json` и се уверете, че необходимите OAuth права са зададени.
4. В **Project Settings > Script Properties** създайте ключ `SENDGRID_KEY` и въведете вашия API ключ на SendGrid.
5. Разположете скрипта като уеб приложение (Deploy > New deployment > Web app) и позволете достъп до всеки, който има връзката.

## Работа

1. Отворете връзката към уеб приложението.
2. Сканирайте баркод в полето **„Сканирай тук“** или въведете ръчно артикулен номер.
3. След приключване на ревизията натиснете **„Край на ревизия“**, за да се създаде файл в Google Drive.
4. При нужда от ППР използвайте бутона **„Въвеждане на ППР“**, попълнете данните и натиснете **„Запиши в ППР“**.
5. В раздел **„Преглед на ревизии“** може да се видят предишни ревизии по магазини.
